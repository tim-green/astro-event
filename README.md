# E-vent: Astro Event Landing Page

## Project Overview

E-vent is a modern and responsive event landing page built with Astro. This project provides a clean and attractive interface to showcase event details, designed to be easily customizable for various types of events.

## Features

- **Responsive Design**: Ensures a seamless experience across devices with a mobile-first approach.
- **Modern Tech Stack**: Built with Astro, Tailwind CSS, and GSAP for animations.
- **Customisable Components**: Modular architecture allows easy customization and extension.
